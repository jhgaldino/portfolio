# Portfolio - Jonathan Galdino

## 📋 Descrição
Portfolio pessoal desenvolvido com HTML, CSS e JavaScript, apresentando habilidades, projetos e experiência profissional. Disponível em português e inglês.

## 🛠️ Tecnologias Utilizadas
- HTML5 / CSS3 / JavaScript
- Bootstrap 5.3
- Font Awesome 5.15.4
- Devicons 2.16 (logos oficiais das tecnologias)
- Inter (Google Fonts)
- AOS (animações de scroll)

## 🌟 Características
- Design responsivo (mobile-first)
- Tema dark/light com persistência via localStorage
- Troca de idioma PT/EN
- Hero com texto animado (efeito digitação)
- Seções: Sobre, Habilidades, Projetos, Experiência, Educação, Contato
- Skills organizadas por categoria com ícones Devicon
- Cards de projetos com tech tags e efeito hover
- Links para GitHub, LinkedIn, WhatsApp e e-mail
